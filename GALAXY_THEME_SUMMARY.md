## 🌟 Key Animations

### **Twinkle** (Stars)
```css
0%, 100% → opacity: 0.5
50% → opacity: 0.8
```

### **Nebula** (Clouds)
```css
0%, 100% → translate(0, 0) rotate(0deg)
50% → translate(-10%, -10%) rotate(5deg)
```

### **Shooting Star**
```css
0% → Start position (invisible)
10% → Fade in
20% → Fade out at end position
```

### **Galaxy Gradient**
```css
Animates background position
Creates flowing color effect
```
